# 📸 Lightbox Viewer

The **Lightbox Viewer** is a simple and interactive web project built with **HTML, CSS, and JavaScript**. It allows users to click on images and view them in a larger, focused overlay without leaving the page.

This improves user experience by keeping everything on one page while highlighting the selected content.

---

## ✨ Features

✔️ Clickable image thumbnails
✔️ Full-size image display in overlay
✔️ Dark background focus effect
✔️ Close button functionality
✔️ Smooth user interaction

---

## 🛠️ Built With

* HTML5
* CSS
* JavaScript 

---

## ⚙️ How It Works

* Images are displayed as thumbnails on the page
* Clicking an image triggers JavaScript
* The lightbox overlay appears with the full image
* Background dims to focus attention
* User can close the viewer to return

---

## 🚀 How to Use

1. Clone the repository: git clone https://github.com/Fikayomi-jpg/Lightbox-Viewer.git
2. Open the project folder: cd Lightbox-Viewer
3. Open `index.html` in your browser
